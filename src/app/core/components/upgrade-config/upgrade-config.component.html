All New Touch, Mouse, and Keyboard Navigation

Heads up: The new version introduces a completely redesigned fullscreen interface with no navigation bar. Please review the input modes table below to get an understanding of the new ways to operate Skip before upgrading. After the upgrade, give yourself time to get comfortable with the changes.

Actions Touch Mouse Keyboard Shortcuts
Show the toolbar Swipe down from the top Scroll up, or tap the top peek strip N/A
Move between pages Swipe left or right Scroll horizontally Ctrl + ←/→ (Left/Right Arrow)
Enter page edit mode Tap the edit button on the toolbar Shift + Ctrl + E
Toggle Fullscreen Tap the fullscreen button on the toolbar Shift + Ctrl + F
Toggle Night mode Tap the night-mode button on the toolbar Shift + Ctrl + N
Note that the words Touch and Tap are synonymous with mouse click.

Migration Limitations

Important: The new widget layout engine is completely different from the previous version. When you upgrade, your configuration will be migrated, but the way pages and widgets are arranged will change significantly.

  • All Skip configuration settings will be migrated.
  • Each v2 Page layout becomes its own v3 page.
  • All widgets and their settings will appear in their new pages.
  • Note: Widget positions, order, and sizes WILL BE LOST. After upgrading, pages will contain small widgets that you must reposition and resize manually.

To upgrade your configuration, follow these steps:

  1. As a safety measure, if you are using Signal K authentication, manually back up your configuration. Make a copy of the 9.0.0.json files found on the Signal K server in the following folders:
    • ~/.signalk/applicationData/users/(username)/kip/
    • ~/.signalk/applicationData/global/kip/
  2. Click the Upgrade button below.
  3. Once the upgrade is complete, Skip will restart and display the migrated pages.
  4. Reposition and resize your widgets as needed.
  5. Once satisfied, delete the 9.0.0.json configuration files located in the previously mentioned folders. They are not needed any more.

For tips and guidance, consult Skip’s in-app Help section. Most common questions are answered there. If you need support, join the Signal K #kip Discord channel.
